Xiaomi Flower Care, Mijia Sensor, Bluetooth добавляем в Domoticz
===============================================

* Скачайте 
* `sudo git clone https://github.com/zver5/ble`
* перейдите в папку
* `cd ble`
* Запустите поиск устройств
* `sudo hcitool lescan`
* 4C:65:A8:D0:45:D0 MJ_HT_V1 = Mijia Sensor
* C4:7C:8D:62:0E:F4 Flower care
